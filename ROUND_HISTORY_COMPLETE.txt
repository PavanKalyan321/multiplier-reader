================================================================================
                          FEATURE OVERVIEW
================================================================================

The Multiplier Reader now includes a COMPLETE ROUND HISTORY SYSTEM that:

1. AUTOMATICALLY TRACKS every game round played
2. STORES comprehensive data for each round
3. DISPLAYS formatted history after each round completes
4. GENERATES statistics across all rounds
5. HANDLES read failures gracefully as round endings

================================================================================
                        WHAT GETS RECORDED
================================================================================

For each completed round:

  Round Number        Sequential ID (1, 2, 3, ...)
  Start Time          When round began (HH:MM:SS format)
  End Time            When round ended (HH:MM:SS format)
  Duration            Total time (in seconds)
  Max Multiplier      Highest value reached during round
  Crash Multiplier    Value when game crashed (usually 0)
  Status              Round outcome (CRASHED, COMPLETED, etc.)
  Event Count         Number of game events in this round

Plus:

  Statistics:
  - Maximum multiplier ever reached (all rounds)
  - Average maximum multiplier per round
  - Average duration per round
  - Total number of crashes
  - Crash rate percentage

================================================================================
                       TECHNICAL DETAILS
================================================================================

NEW CLASS: RoundSummary
Located in: game_tracker.py
Purpose: Store all data for one completed round
Data: 8 fields
Method: to_dict() converts to formatted display

NEW TRACKING ARRAY: round_history
Located in: GameTracker class
Contains: List of RoundSummary objects
Access: game_tracker.round_history

NEW COUNTER: round_number
Located in: GameTracker class
Purpose: Sequential ID for each round
Increments: Every time a round completes

DETECTION:
- Round Start: When multiplier = 1.0x and status = 'STARTING'
- Round End: When multiplier drops below crash_threshold (default 0.5)
- Auto-Save: Immediately saved to round_history array

READ FAILURE HANDLING:
- Failed OCR reads no longer break the system
- Checked as potential round-end signals
- If round was active, logs it
- If no round active, continues normally
- Application stability maintained

================================================================================
                          HOW TO USE IT
================================================================================

AUTOMATIC (Default):
  1. Run: python main.py
  2. Play rounds normally
  3. History displays automatically after each round
  4. Full history shown on exit (Ctrl+C)
  NO ACTION NEEDED - it just works!

PROGRAMMATIC (In custom code):
  # Get all rounds
  all_rounds = app.game_tracker.get_round_history()

  # Get last 5 rounds
  recent = app.game_tracker.get_round_history(limit=5)

  # Get statistics
  stats = app.game_tracker.get_round_statistics()
  print(stats['avg_max_multiplier'])

  # Get formatted table
  table = app.game_tracker.format_round_history_table()
  print(table)
